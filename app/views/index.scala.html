Welcome to the Template Site!

The Template Site is a project created by Tetrao, a technology startup company specialized in Internet intelligent process automation. Our mission is enabling connections between people and businesses radically faster in an efficient and cost-effective way.

This is an internal project that we decided to publish it to promote the use of a lot of open source projects we love and we use every day and make our job more easy.

Template Site is an skeleton of a web application based on:

  • Scala as programming language
  • SBT as build tool
  • Play Framework as web application framework
  • Play2-auth as authentication module for Play
  • PostgreSQL as database
  • Slick as database access layer
  • Slick-pg extending slick for support PostgreSQL data types
  • Foundation as front-end framework

Check the build.sbt file to get the different versions used. Our intention is to keep updated this project as new versions of this software are published.
